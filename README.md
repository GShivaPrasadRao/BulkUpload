
# Bulk Upload(CSV & Excel)

This application helps in uploading the data into Database
which is in CSV or Excel format.

## Tech Stack

**Server:** Spring Boot, Apache POI(Excel),Apache Commons(CSV)

**Database:** MySQL

**Tools** : Postman
